# Suggestions for Freshers

## Resume Tips
- Keep it one page, ATS-friendly
- Quantify impact (e.g., "Reduced API response time by 40%")
- Link to GitHub, LinkedIn, portfolio
- Tailor keywords to each job description

## Getting Started
- Focus on fundamentals: DSA, OOP, DBMS, OS, Networking
- Build projects, not just follow tutorials
- Learn Git & GitHub early

## Skills to Develop
- **Languages:** Java / Python / JavaScript (pick one, master it)
- **Web:** HTML, CSS, JS → any framework (React preferred)
- **Backend:** Node.js / Spring Boot / Django
- **Database:** SQL (MySQL/PostgreSQL) + one NoSQL (MongoDB/Redis)
- **DevOps basics:** Linux, Docker, CI/CD concepts

## Projects to Build
1. Todo app with auth (full stack)
2. Blog platform with CRUD
3. E-commerce API with payment integration
4. Real-time chat app
5. Portfolio website

## Job Prep
- Solve 200+ DSA problems (LeetCode / Codeforces)
- Practice system design basics
- Build a strong LinkedIn & GitHub presence
- Apply to 10+ companies daily
- Prepare behavioral answers (STAR method)

## Mindset
- Consistency > Intensity
- Learn in public — share what you build
- Don't compare; focus on your growth
- Rejections are redirections
